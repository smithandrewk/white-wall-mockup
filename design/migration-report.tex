\reportsection{What You Unlock by Leaving Squarespace}

Squarespace is a walled garden. On a standard web stack, you get access to the entire modern web ecosystem. Here are three categories of tools that become available:

\reportsubsection{Analytics: PostHog (Free, Replaces Squarespace Analytics)}

Squarespace gives you basic page views. PostHog gives you:

\begin{multicols}{2}
\begin{itemize}
  \item \textbf{Session recordings}: watch exactly how visitors navigate. See where they hesitate, what they click, where they drop off before booking.
  \item \textbf{Heatmaps}: visual overlay of clicks and scroll depth on each page.
  \item \textbf{Funnels}: track the path from landing to booking. ``80\% see the gallery, but only 10\% click Book Now.''
  \item \textbf{A/B testing}: test two page versions and measure which converts better.
  \item \textbf{Feature flags}: roll out changes to 10\% of visitors, check data, then go to 100\%.
\end{itemize}
\end{multicols}

\callout{%
  \textbf{Free tier:} 1 million events/month, 5,000 session recordings/month. More than enough for WhiteWall's traffic. This level of analytics is \emph{not available on Squarespace at any price tier}.%
}

\reportsubsection{Automations: Zapier, Make, and n8n}

Connect your site events to anything. Examples relevant to WhiteWall:

\begin{multicols}{2}
\begin{itemize}
  \item New booking $\to$ auto-send confirmation email with studio access codes, parking info, prep checklist
  \item New booking $\to$ add row to Google Sheets for revenue tracking per location
  \item Form submission $\to$ create a lead in your CRM (or a Google Sheet)
  \item New booking $\to$ send SMS reminder 24 hours before the session
  \item Membership signup $\to$ auto-add to a private email list
  \item Low booking week $\to$ auto-post Instagram story with availability
\end{itemize}
\end{multicols}

\callout{%
  Zapier free tier: 100 tasks/month. Make.com: even more generous. n8n: fully free and self-hosted. On Squarespace, you're limited to their handful of built-in integrations. Off Squarespace, your site can talk to \emph{literally anything with an API}.%
}

\reportsubsection{AI-Powered Features (Claude via Vercel)}

With Vercel, you can add server-side API routes that run AI models. This is where a modern stack truly separates from Squarespace:

\begin{multicols}{2}
\begin{itemize}
  \item \textbf{AI booking assistant}: a chat widget that answers questions about studios, pricing, and availability, then walks visitors through booking. Powered by Claude, trained on your content.
  \item \textbf{Smart inquiry routing}: AI reads contact form submissions and auto-categorizes them (booking inquiry, gear rental, event inquiry, partnership pitch), then routes to the right person or sends an auto-reply.
  \item \textbf{Auto-generated social captions}: upload a gallery photo, AI writes an Instagram caption with hashtags matching your brand voice.
  \item \textbf{Dynamic FAQ}: instead of a static page, a conversational interface that answers any question about the studios.
  \item \textbf{Personalized landing pages}: returning visitors see content tailored to their location preference (Taylor's Mill vs.\ Powdersville) based on previous browsing.
\end{itemize}
\end{multicols}

\accentcallout{%
  \textbf{Cost:} Claude API is pay-per-use. A booking assistant handling 100 conversations/month costs roughly \textbf{\$2--5/month} in API fees. Compare that to a \$49/mo chatbot SaaS. \textbf{None of this is possible on Squarespace} -- you can't run server-side code, call APIs, or add custom logic.%
}

\reportsection{Risks \& Considerations}

\begin{itemize}
  \item \textbf{Editing without code:} If Drew/Max frequently edit page content, they'll need a CMS (Decap, Tina) or someone to make HTML changes. Squarespace's visual editor is the one thing that's hard to replace for free.
  \item \textbf{Email:} You use Google Workspace (MX records), not Squarespace email. No email migration needed.
  \item \textbf{SEO:} Redirect old Squarespace URLs to new ones. Squarespace uses paths like \texttt{/about-taylors-mill-location} which we can keep or redirect.
  \item \textbf{Domain transfer:} whitewallstudios.co needs to be transferred from Squarespace. Standard process, \textasciitilde10 minutes + up to 5 day transfer window.
\end{itemize}
